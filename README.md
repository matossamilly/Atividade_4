# Atividades - Lista 1: Algoritmos e Lógica de Programação 🚀

Este repositório contém a resolução da **Lista 1** de atividades práticas de lógica de programação. O foco principal é exercitar a construção de algoritmos para situações do cotidiano através de **fluxogramas** e **pseudocódigos**.

---

## 📌 Objetivo
Desenvolver a capacidade de abstração e o pensamento sequencial, decompondo tarefas complexas em passos lógicos, finitos e claros.

---

## 📂 Atividades Desenvolvidas

A lista consiste na resolução de 5 desafios lógicos:

1.  **Ir ao Trabalho 🏃‍♂️:** Sequência desde o despertar até a chegada ao destino, considerando variações de transporte.
2.  **Troca de Pneu 🚗:** Procedimento de segurança e mecânica básica para substituição de um pneu furado.
3.  **Troca de Lâmpada 💡:** Processo de manutenção doméstica focando em segurança e teste de funcionamento.
4.  **Pegar um Ônibus 🚌:** Lógica de deslocamento urbano, envolvendo esperas e tomadas de decisão.
5.  **Preparar um Bolo Simples 🍰:** Algoritmo culinário com foco em ordem de ingredientes e tempo de execução.

---

## 🧠 O que aprendi com este projeto

A elaboração desta lista foi fundamental para consolidar conceitos base da computação:

* **Pensamento Algorítmico:** Transformar tarefas do mundo real em uma sequência lógica que não permite ambiguidades.
* **Decomposição de Problemas:** A capacidade de quebrar um problema grande em sub-tarefas menores e mais simples de resolver.
* **Estruturas de Decisão:** Aplicação de condicionais (**SE / SENÃO**) para lidar com diferentes cenários (ex: se o pneu estiver furado, se o ônibus passar, etc).
* **Padronização Visual:** Uso correto da simbologia de fluxogramas para representar processos, decisões e pontos de início/fim.
* **Visão Crítica:** Entender que a lógica de programação é, antes de tudo, uma ferramenta para estruturar a resolução de qualquer problema.
